# Transliteration FULL name bot
Бот который переводит ФИО на английский язык

**[@Transliteration_FULL_name_bot](https://web.telegram.org/a/#8646734507)**

## Функциональность

- ✅ Принимает ФИО на русском языке
- ✅ Проверяет корректность ввода (только кириллица, пробелы и дефисы)
- ✅ Транслитерирует в соответствии с Приказом МИД России от 12.02.2020 № 2113
- ✅ Логирует все действия в файл
- ✅ Работает в Docker контейнере
- ✅ Обрабатывает двойные фамилии (Анна-Мария → Anna-Mariia)# Transliteration-FULL-name-bot
